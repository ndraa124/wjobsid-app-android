# WJobsID Android

A simple android app using kotlin and node js

## Requirements

1. Android 23+

## Features:

```
1. Login
2. Registrasi
3. List Engineer
4. List Project (Company)
5. List Hiring (Engineer)
6. List Portfolio
7. List Experience
8. List Skill
9. Tambah Skill
10. Edit Skill
11. Hapus Skill
12. Tambah Portfolio
13. Edit Portfolio
14. Hapus Portfolio
15. Tambah Experience
16. Edit Experience
17. Hapus Experience
18. Tambah Project
19. Edit Project
20. Hapus Project
21. Approve/Reject Hiring (Engineer)
22. Edit Profile (Engineer)
23. Edit Profile (Company)
24. Lupa Password
25. Upload Image (Portfolio, Project, Foto Profile)
```
